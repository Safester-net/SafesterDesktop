Aide Nouvel Utilisateur

Paramètres Internet sur votre PC ou Mac

        
  • Utilisateurs Windows: le programme "java.exe" doit être autorisé à passer des appels http. Vérifiez les paramètres de votre pare-feu et de votre antivirus local afin d'autoriser "java.exe" à communiquer via http.     
    •     
  • UtilisateursMac OS et Linux: le processus "java" doit être autorisé à passer des appels http. Vérifiez vos paramètres antivirus et de pare-feu local afin d'autoriser "java" à  communiquer via http.

A propos des paramètres de proxy:
Safester détectera automatiquement vos paramètres de proxy. Aucune configuration manuelle n'est donc nécessaire... Si vous le souhaitez, vous pouvez forcer votre configuration à l'aide de l'élément "Paramètres de proxy" du menu "Paramètres".

Utilisateurs professionnels: paramètres du pare-feu

Le pare-feu de l'entreprise doit autoriser les connexions https entrantes et sortantes aux serveurs Web Safester avec URL qui commencent par: https://www.runsafester.net.

Veuillez vérifier auprès de votre administrateur système que le pare-feu de votre entreprise accepte le trafic http entre votre URL du PC ou du Mac et du serveur Web Safester.

Création du compte

Cliquez simplement sur le bouton "Nouvel utilisateur ? Créer un Compte". La fenête Créer un compte vous demandera le nom, l'email et la passphrase à utiliser pour créer immédiatement votre compte Safester. Vous pourrez alors envoyer des messages chiffrésà qui que ce soit.
