This window allows you to import contacts from a CSV text file (Comma Separated Values).

The files must contain in every line the 2 or 3 following fields separated by a comma or a semicolon:
  • 2 fields: Email address, Full Name.
  • 3 fields: Email Address, First Name, Last Name.

Please select the CSV file to use by clicking "Next>".
You can also drag & drop the file on the window.
