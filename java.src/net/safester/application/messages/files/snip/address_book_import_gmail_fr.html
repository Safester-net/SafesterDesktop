Pour importer vos Contacts Gmail, vous devez autoriser l'application à accéder en lecture à vos Contacts puis copier/coller un Code qui sera fourni par Google.

A cet effet, cliquez sur le lien "Obtenir un Code" ci-dessous. Ceci :
  • ouvrira une page Google d'accès à votre compte Gmail,
  • vous présentera ensuite une demande d'autorisation d'accès en affichage à vos Contacts,
  • puis affichera le Code à copier/coller dans le champ texte ci-dessous.

Une fois le Code renseigné, cliquez sur "Suivant >".
Vos Contacts Gmail seront alors téléchargés et prêts à importer après vérifications.
