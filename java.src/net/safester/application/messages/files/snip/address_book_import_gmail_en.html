To import your Gmail Contacts, you must give the application a read access to your Contacts and copy / paste a code that will be provided by Google.

For this, click on the "Get a Code" below. This:
  • will open a Google page to access your Gmail account,
  • will then present you a read authorization access to your Contacts,
  • will show the code to copy / paste into the text field below.

Once the Code filled, click "Next>".
Your Gmail contacts will then be downloaded and ready to be imported after verification.
