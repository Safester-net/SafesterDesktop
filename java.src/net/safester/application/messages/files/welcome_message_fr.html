Bienvenue sur Safester!

Safester est la solution de messagerie sécurisée facile à utiliser, facile à partager.

Pour envoyer un email confidentiel à l'un de vos contacts, composez simplement un message sur son adresse email habituelle.
S'il n'est pas encore enregistré dans le système Safester, il recevra une invitation. Il recevra votre message dès qu'il s'enregistrera auprès de Safester.

Si vous avez des questions ou des remarques sur Safester, veuillez nous contacter à l'adresse contact@safester.net.
