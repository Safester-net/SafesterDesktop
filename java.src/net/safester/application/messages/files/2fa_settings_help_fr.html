Paramètres 2FA

        
  • Installez et ouvrez sur votre téléphone portable une application 2FA compatible avec les normes HOTP / TOTP. (andOTP, FreeOTP ,Google Authenticator, LastPass Authenticator, etc.).
    •     
  • Dans l'application Safester Desktop, accédez au Menu 2FA ==> QR Code du Compte. Ceci créera votre compte 2FA s'il n'existe pas déjà, puis affichera le QR Code à analyser.
    •     
  • Scannez le QR Code affiché avec votre téléphone portable et une App 2FA.
    •     
  • Fermez la fenêtre "QR Code du Compte" lorsque le scan est terminé.
    •     
  • La fenêtre "Statut d'activation 2FA" vous permettra d'activer / désactiver l'entrée de validation de code 2FA pour les prochaines connexions sur Safester.

Avertissements
        
  • N'activez pas l'authentification 2FA si vous n'avez pas scanné le QR Code avec votre application mobile 2FA: vous ne pourriez plus vous connecter et vous authentifier sur Safester.
    •     
  • Nous vous recommandons d'utiliser une application 2FA prenant en charge la sauvegarde des informations de compte.
    •     
  • Assurez-vous de bien sauvegarder les informations relatives à vos comptes mobiles 2FA Safester.
