New User Help

Internet Settings on your PC or Mac

  • Windows users:
    "java.exe" program must be authorized to make http calls. Check your antivirus & local firewall settings in order to authorize "java.exe" to communicate through http.

  • mac OS and Linux users:
    "java" process must be authorized to make http calls. Check your antivirus & local firewall settings in order to authorize "java" to communicate through http.

About Proxy Settings:
Safester will automatically detect your proxy settings, so there is no need for any manual configuration. If wanted, you can force your configuration using the "Proxy Settings" Item in the "Settings" Menu.

Corporate users: firewall settings

The corporate firewall must authorize inbound and outbound https connections to the Safester Web Server with URLs that start with: https://safester.net.

Please check with your System Administrator that your corporate firewall accept the http traffic between your PC or Mac and the Safester Web Server URLs.

Account Creation

Just click on the "New User? Create New Account" button. The Create Account window will ask you the name, email and passphrase to use in order to immediately create your Safester Account. You will then be able to send encrypted messages to anyone.
