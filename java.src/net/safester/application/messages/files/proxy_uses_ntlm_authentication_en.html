Proxy uses Microsoft NTLM authentication
If your proxy uses Microsoft NTLM proxy authentication, check the box and provide the Workstation and Domain values.

Safester supports Microsoft NTLM but can not detect automatically if your corporate proxy requires NTLM authentication.
Please check with your System Administrator.

Safester supports both NTLMv1, NTLMv2 and NTLMv2-Session.
