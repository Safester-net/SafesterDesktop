Compose Message

Composition de votre message

Composez et envoyez votre message Safester comme vous le feriez pour tout courrier électronique ordinaire:
        
  • Ajoutez un ou plusieurs destinataires dans les zones "A" ou "Cc". Il suffit d'utiliser des adresses électroniques existantes de type email@domain.net.
    •     
  • Vous pouvez joindre tout type de fichier à votre message. Vous pouvez glisser-déposer les fichiers de votre bureau sur la fenêtre dédiée.
    •     
  • Utilisez l'éditeur de texte enrichi pour composer le corps du message.
    •     
  • Option "No Foward" :         
    Si cette option est sélectionnée (bouton enfoncé), les destinataires ne pourront pas transférer le message avec Safester.
    •     
  • Option "No Print" :         
    Si cette option est sélectionnée, les destinataires ne pourront pas imprimer le message.
    •     
  • Option "Envoyer une notification anonyme" :         
    Les destinataires
    seront informés de votre message dans leur boîte aux lettres habituelle. Si l'option est sélectionnée, la notification sera anonyme et votre nom et votre adresse de messagerie n'apparaîtront pas.

Le corps du message et les fichiers joints seront chiffrés sur votre PC, puis téléchargés sur notre serveur sécurisé Safester.
Le cryptage est effectué avant l'envoi, donc aucune information claire ne quittera votre PC.

Remise de votre message chiffré

La livraison dépend du statut Safester de chaque destinataire.
        
  • Si le destinataire possède déjà un compte Safester:         
    il recevra une notification sur son adresse email. Une fois connecté à son compte Safester, il pourra déchiffrer et lire votre message et les fichiers joints qui seront déchiffrés en toute sécurité sur son PC.
    •     
  • Si le destinataire n'a pas de compte Safester:         
    il recevra sur son adresse email une invitation à créer un compte gratuit Safester.         
    Ne vous inquiétez pas! L'utilisation du compte créé est entièrement gratuite et disponible sur tous les principaux systèmes d'exploitation de bureau. (Windows, Mac OS et Linux), Android et iOS.         
    Le destinataire récupérera et déchiffrera en toute sécurité votre message chiffré lors de la première connexion. Il sera alors en mesure de répondre à votre message et de vous renvoyer des messages chiffrés, à nouveau sans aucun frais.
