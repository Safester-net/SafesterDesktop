Importer des Contacts

Vous pouvez importer des contacts :
  • A partir de Microsoft Outlook Office (Windows uniquement)
    L'installation d'Outlook est automatiquement détectée sur votre PC.
    Il suffit alors de sélectionner votre dossier Contacts à utiliser et d'indiquer l'ordre d'insertion des noms et prénoms.  
    N.B: Outlook doit rester ouvert pendant toute la durée de l'import.

  • A partir d'un compte Gmail.
    Il vous suffit de suivre les instructions qui vous demanderont d'autoriser temporairement l'application à accéder à vos Contacts en lecture, puis de copier/coller un code d'autorisation.

  • A partir d'un fichier au format CSV.
    Les caractères ";" ou "," sont utilisés comme séparateur.
