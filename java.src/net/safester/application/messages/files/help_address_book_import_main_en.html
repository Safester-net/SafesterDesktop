Importing Contacts

You can import contacts:
  • From a CSV file
    The characters ";" or "," are used as the separator.

  • From a Gmail account
    Simply follow the instructions which will ask you to temporarily allow the application to access your Contacts in read only, and then copy / paste an authorization code.

  • From Microsoft Office Outlook (Windows only)
    Outlook is automatically detected on your PC.
    You simply need to select the Contacts folder to use and specify the order of insertion of last names and first names. -
    N.B.: Outlook must remain open for the duration of the import.
