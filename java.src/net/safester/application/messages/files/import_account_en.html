This tool allows you to import an existing Safester account into your current account.
WARNING: This operation cannot be undone.

  • All messages of imported account will be imported in a "Imported messages" folder.
  • All email addresses linked to the imported account will become aliases for your account.
  • The imported account will be deleted and will no longer be accessible.
As the import process may take a while, you will be notified by email when the import will be completed.
