Safester utilise de la cryptographie forte avec le standard OpenPGP.

L'algorithme par défaut et les clés longues utilisés (ElGamal - 2048 bits / AES - 256 bits) offrent une très grande sécurité: les données chiffrées sont considérées comme indéchiffrables par tout attaquant.
